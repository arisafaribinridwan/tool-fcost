Ringkasan Layout

  • Window utama dibagi menjadi 2 kolom.
  • Kolom kiri bersifat scrollable dan fokus pada input serta konteks sesi.
  • Kolom kanan fokus pada aksi proses dan pemantauan log runtime.
  • Ukuran window saat ini fixed di 1280x720.

Daftar Elemen Saat Ini

  • Header: judul aplikasi, subtitle, tombol Pengaturan Job.
  • Source Input: label source, tombol Clear Source, panel source aktif, tombol Pilih Source.
  • Job Aktif: dropdown job, tombol Refresh, tombol Settings, info sesi terakhir, detail job aktif, tombol Start New Session.
  • Execute: tombol Execute, tombol Buka Folder Outputs.
  • Process Log: judul log, label Runtime monitor, textbox log.
