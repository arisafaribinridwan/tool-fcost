Learning path

Belajar programming dari project nyata ini

Jika kamu baru masuk dunia pemrograman, project ini bagus untuk belajar karena menggabungkan UI, file system, data processing, config, Excel, testing, dan packaging. Jangan mencoba memahami semuanya sekaligus; ikuti urutan bertahap di halaman ini.

Roadmap belajar 6 tahap

  1. Dasar Python. Pahami variable, function, list, dict, if/else, loop, exception, dan import.
  2. File dan path. Pelajari pathlib.Path, ekstensi file, folder runtime, dan kenapa path traversal berbahaya.
  3. Data tabular. Pelajari pandas DataFrame sebagai “tabel Excel di memory”. Coba baca CSV kecil, filter baris, merge lookup, group by.
  4. YAML config. Pahami bahwa YAML adalah cara menulis aturan tanpa mengubah kode Python.
  5. UI desktop. Lihat bagaimana button, label, dialog, thread, dan queue dipakai di CustomTkinter.
  6. Testing. Pelajari pytest dari test kecil yang membuat file sementara, menjalankan service, lalu mengecek output.

Latihan kecil yang aman

  1. Buka tests/test_source_service.py, pahami test validasi source, lalu tambahkan test untuk file kosong jika belum ada.
  2. Buat CSV kecil berisi kolom kode_produk, qty, dan harga. Coba baca dengan pandas di Python shell.
  3. Duplikasi test pipeline happy path dan ubah nama sheet output menjadi Ringkasan. Jalankan hanya test itu.
  4. Buat config klasik sederhana yang hanya memilih beberapa kolom tanpa master lookup.
  5. Tambahkan satu kolom formula sederhana di config test, misalnya total = qty * harga.

Cara membaca source code tanpa pusing

Mulai dari alur paling luar, bukan dari detail terdalam. Buka run.py, lalu app/__init__.py, lalu app/ui/main_window.py bagian add_log_event(), kemudian masuk ke pipeline_service.py. Setelah paham pipeline, baru baca recipe_service.py atau transform_service.py.

Untuk file besar seperti recipe_service.py, jangan baca dari atas sampai bawah dalam satu waktu. Cari function utama seperti execute_step_recipe(), lihat dispatch step, lalu baca satu step yang sedang kamu pelajari.
